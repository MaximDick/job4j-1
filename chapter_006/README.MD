## <a href="https://github.com/inflatone/job4j/tree/master/chapter_006/src/main/java/ru/job4j/bomberman">Бомбермен</a>
##### Реализовать бизнес-логику (без графики) игры Бомбермен с использованием многопоточности.
1. Есть поле, состоящее из клеток, на кот. и происходит действие игры.
2. Главный герой может передвигаться по нему при условии, что клетка свободна.
3. Чудовища передвигаются автоматически каждые 1 сек. в произвольном направлении.
При невозможности совершить ход (клетка занята) подождать 0.5 сек. и совершить ход в новом направлении.
4. Игра считается оконченной, когда главный герой сталкивается с чудовищем в одной клетке.

## <a href="https://github.com/inflatone/job4j/tree/master/chapter_006/src/main/java/ru/job4j/cache">Неблокирующий кеш</a>
##### Реализовать работу неблокирующего кеша на основе версионирования и ConcurrentHashMap.
1. Необходимо сделать кеш для хранения моделей. в кеше должны быть методы CRUD.
2. Обеспечить потокобезопасность кеша.
3. В кеше должна быть возможность проверять актуальность данных (Optimistic concurrency control).
